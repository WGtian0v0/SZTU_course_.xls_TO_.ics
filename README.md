# 将技术大学的课表的.xls文件转换成.ics日历文件
本人有列计划的习惯，但是由于系统日历无法显示课表，只能一个个手动添加。市面上的其他方案要么不够完善，要么收费，无奈之下遂写了这个程序。

这个程序可以将深圳技术大学的学生课表转换成.ics日历文件，方便导入到手机或其他设备的日历中。

（如果教务系统课表有变动，要手动再添加一遍哦＜(´⌯  ̫⌯`)＞）
